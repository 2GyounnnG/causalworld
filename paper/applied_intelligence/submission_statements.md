# Submission Statements

## Code Availability Statement

The code used for training, analysis, aggregation, auditing, and figure generation can be made available upon review or publication, subject to repository cleanup and any venue-specific anonymity requirements. The analysis artifacts used for this manuscript are organized under `analysis_out`, and the paper package is organized under `paper/applied_intelligence`.

## Data Availability Statement

The experiments use public benchmark data and generated result artifacts. The molecular experiments use the rMD17 benchmark, and the Wolfram experiments use locally generated synthetic relational-dynamics outputs. Processed aggregate CSV files and analysis reports used for the manuscript are included in the repository artifacts.
